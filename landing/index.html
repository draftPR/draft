How it Works

From idea to merged code in minutes

🎯

Describe the goal

Point at your codebase. Alma analyzes the code and generates scoped, accurate tickets.

⚡

Agents execute

AI agents pick up tickets and write real code in isolated git worktrees. Run many in parallel.

👀

You review

GitHub-style diff viewer with inline comments. Approve, request changes, or let agents iterate.

🚀

Merge and ship

One click to merge to main. Or auto-create a GitHub PR. Full lifecycle, one tool.

Features

Everything you need to ship with agents

🧠

Codebase Analysis

Point at a repo and Alma generates tickets from actual code. No hallucinated functions.

🔀

Git Worktree Isolation

Each ticket runs in its own worktree. Agents never conflict with each other or your main branch.

💬

Code Review Built In

Full diff viewer with syntax highlighting and inline comments. No context switching.

🤖

Multi-Agent Support

Run Claude Code, Codex, or any agent. Assign by capability. Track performance per agent.

📡

REST API and MCP

Full API for programmatic access. MCP server so agents can discover and work tickets natively.

🔄

GitHub Sync

Import issues, export PRs. Two-way sync keeps your existing workflow intact.

📊

Live Dashboard

See what every agent is working on. Streaming logs, cost tracking, completion estimates.

🔐

Autonomy Controls

Auto-approve low-risk changes, require review for sensitive code. You set the guardrails.

🏠

Self-Hostable

SQLite backend, zero dependencies. Run on your machine with full control. Or use cloud.
